# Sport 18 - Premium Sports Streaming Dashboard

A high-performance, responsive sports streaming dashboard that dynamically fetches live channel data.

## Features
- **Dynamic Data**: Automatically fetches channels from the latest source.
- **Premium Design**: Modern dark theme with glassmorphism and smooth animations.
- **Real-time Search**: Quickly find your favorite channels.
- **Responsive**: Works seamlessly on mobile, tablet, and desktop.

## How to use
1. Clone the repository.
2. Open `index.html` in your browser (preferably via a local server to avoid CORS issues).

## Credits
All channel content is fetched from external sources.
